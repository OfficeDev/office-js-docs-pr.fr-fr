# <a name="formsettings-element"></a>Élément FormSettings

Spécifie l’emplacement source et les paramètres de contrôle de votre complément de messagerie.

**Type de complément :** messagerie

## <a name="syntax"></a>Syntaxe

```XML
<FormSettings>
   ...
</FormSettings>
```
