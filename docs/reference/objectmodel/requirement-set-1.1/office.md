# <a name="office-mailbox-requirement-set-11"></a>Office (ensemble de conditions requises de boîte aux lettres 1.1)

L’espace de noms Office fournit des interfaces partagées qui sont utilisées par des compléments dans toutes les applications Office. Cette liste documente uniquement les interfaces utilisées par des compléments Outlook. Pour obtenir une liste complète des espaces de noms Office, consultez la page relative à l’[interface API commune](/javascript/api/office?view=outlook-js-1.1&preserve-view=true).

|---|---|---|
|`Succeeded`| String|L’appel a réussi.|
