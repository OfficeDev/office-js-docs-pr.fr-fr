Pour exiger l’ensemble d’API d’identité 1.3 dans votre code de Outlook, vérifiez s’il est pris en charge par `isSetSupported('IdentityAPI', '1.3')` l’appel. Sa déclaration dans le manifeste du Outlook n’est pas prise en charge. Vous pouvez également déterminer si l’API est prise en charge en vérifiant qu’elle n’est pas `undefined`. Pour plus d’informations, consultez [Utilisation des API d’un ensemble de conditions requises ultérieure](../reference/requirement-sets/outlook-api-requirement-sets.md#using-apis-from-later-requirement-sets).
